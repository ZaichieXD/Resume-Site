Games I Made

These are the games I made over the years

Tap The Circle

Tap the Circle is an mobile arcade game where a circle appears anywhere and tapping it makes it disappear, but be aware that there is a timer and failure to tap the circle will make you lose a life. this is the first game where I implemented Unity Ads.

The Way Out

The Way Out is a puzzle game and the first game I published. Your goal is to escape the maze by placing the boxes on the buttons. You will need to escape before the time runs out failing to exit is Game Over.

Sky Run

Sky Run is a Endless Runner Game made with GDevelop. It's the first game that I made to join a game jam. The Game is not complete and has lots of errors and mistakes. It was my first time using GDevelop even though this game has flaws and failed, I had fun creating this.
